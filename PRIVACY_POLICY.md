# Privacy Policy for PBI Lineage Explorer (Table)

**Last Updated:** March 2026

## 1. Information Collection
The PBI Lineage Explorer (Table) custom visual for Microsoft Power BI ("the Visual") does **not** collect, transmit, distribute, or sell any of your personal data or your organization's data to third-party services or external servers.

### 1.1 In-Memory Processing
All data processing performed by the Visual happens entirely within the memory of the client's web browser or the local Power BI Desktop application.

### 1.2 Telemetry
The Visual does not include any analytics or telemetry tracking scripts that send data externally.

## 2. Information Usage
The Visual uses the data explicitly provided to it through the Power BI field wells (such as Source Node ID, Target Node ID, Refresh Status, Workspace, etc.) strictly for the purpose of rendering the interactive lineage visualization on your screen.
